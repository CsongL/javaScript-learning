# Element和Node的区别
Element继承于Node, 可以使用Node上面的方法, 并且Element对象自身扩展了很多属性
Element是一种特殊的Node， Node还有其他类型特殊的Node

Element是子类， Node是父类， 所以 元素一定是节点， 而节点不一定是元素

在DOM树中，节点的类型:
1. 文档是文档节点
2. HTML元素是元素节点
3. HTML属性是属性节点
4. 文本是文本节点
5. 注释是注释节点

常见的节点类型：
1. Document_Node
2. Text_Node
3. Attribute_Node
4. Element_Node
5. Comment_Node
